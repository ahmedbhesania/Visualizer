# 📊 Sales Data Analyzer (Python Project)

## 🚀 Overview
Sales Data Analyzer is a Python-based command-line tool that helps users perform **data analysis and visualization** on CSV datasets.

It provides features like:
- Data exploration
- Handling missing values
- Data manipulation
- Statistical analysis
- Data visualization (graphs & charts)

---

## 🛠️ Tech Stack
- Python 🐍
- Pandas
- NumPy
- Matplotlib
- Seaborn

---

## 📂 Features

### 🔍 Data Exploration
- View first & last rows
- Check column names
- Inspect data types
- Get dataset info

### 🧹 Data Cleaning
- Show missing values
- Fill missing values (mean or custom)
- Drop missing rows

### ⚙️ Data Operations
- Add new columns
- Multiply numeric columns
- Group data by categories

### 📈 Statistical Analysis
- Summary statistics
- Standard deviation
- Variance

### 🔢 Aggregations
- Sum
- Mean
- Count

### 📊 Pivot Table
- Create pivot tables using selected columns

### 📉 Data Visualization
- Bar Chart
- Line Plot
- Scatter Plot
- Pie Chart
- Histogram
- Box Plot

### 💾 Save Plots
- Save generated plots as image files
